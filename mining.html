Mining for beginners

On this page you will find all the informations you need to start mining BitcoinZ.

As you already know BitcoinZ is based on Equihash algorithm. Every mining softwares are not able to mine cryptocurrencies based on Equihash algorithm. So, because we are nice people, we give you them on a BitcoinZ platter !

Windows GPU Miners

Windows NVIDIA - Dstm Miner

Click Here to see/hide tutorial

zm.exe --server 'server' --port 'port' --user 'WalletAddress'.'WorkerName' --pass x

Example :

zm.exe --server bitcoinz.xyz --port 3035 --user t546ezf4ezzdfsfsSfEZerdd.rig1 --pass x
